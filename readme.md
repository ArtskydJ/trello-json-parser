trello-json-parser
==================

#Usage

```
$ npm install -g trello-json-parser
$ trello-json-parser 6xbOmILH
https://trello.com/b/6xbOmILH.json
{
  "my first list": [
    "a card",
    "another card"
  ],
  "another list": [
    "moar cards",
    "cards,",
    "Cards,",
    "CARDS!!!"
  ]
}
```
